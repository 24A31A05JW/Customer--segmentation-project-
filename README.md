Customer Segmentation Using K-Means Clustering
Overview
This project segments customers based on their Annual Income and Spending Score using the K-Means Clustering algorithm.

Technologies Used
Python
Pandas
Matplotlib
Scikit-Learn
Features
Data preprocessing
Feature scaling
Elbow Method for optimal cluster selection
K-Means clustering
Customer segmentation visualization
Dataset
The dataset contains:

Customer ID
Gender
Age
Education
Marital Status
Annual Income (k$)
Spending Score (1-100)
Results
The Elbow Method identified 5 optimal customer clusters.

Customer Segments
VIP Customers
Regular Customers
Budget Customers
Potential Customers
High-Spending Customers
